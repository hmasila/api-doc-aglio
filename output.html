Get Data
GET/resources

This is endpoint serves to get reviews from API.

You must fill in the following parameters

  • api_user (string) - Get from API Key Generator 268 characters

  • user_accounts (string) - As many user accounts as necessary with a | separator

As with every good action it should return a response. A response always bears a status code. Code 200 is great as it means all is green. Response should also include some data from Amazon.
